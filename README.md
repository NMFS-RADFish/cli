# System Requirements

- Node.js v20+
- `tar` v3.3+

# Creating a project

```
npx @nmfs-radfish/create-radfish-app my-pwa
```

Provide the cli the name of your new project. A new project will be created using the [default boilerplate template](https://github.com/NMFS-RADFish/boilerplate).
